
# Security Policy

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of the Manufacturing Intelligence Platform seriously. If you believe you have found a security vulnerability, please report it to us by emailing **security@mip-platform.example.com**.

**Do not open a public issue.**

When reporting a vulnerability, please include:
- A description of the issue.
- Steps to reproduce (or a proof-of-concept script).
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Security Principles

- **Zero Trust**: No internal component is trusted without verification.
- **Encryption**: All data in transit uses TLS 1.3.
- **Audit Trails**: All administrative actions are logged to an append-only audit table.
